**Maâ€˜een** is an academic awareness project that aims to simplify and present the concept of an **educational virtual application** that serves children and youth with disabilities (hearing impairment, visual impairment, autism, learning difficulties).  

The project focuses on creating a **comprehensive graphic campaign** that includes:

- Designing **complementary educational products** (flash cards, activity book, puzzle/mockups).
